# Forge Project Memory

- Forge Harness is a tutorial-driven TypeScript project for building a coding agent harness from scratch.
- Use `Part 1: Core Harness` and `Part 2: Scale & Extensions` as course part names.
- Use the five Forge layers exactly: `L1 Loop & Execution`, `L2 Governance & Action Boundary`, `L3 Context & Knowledge`, `L4 State, Evidence & Reliability`, `L5 Coordination & Scale`.
- Runnable tutorial chapters follow: `问题 -> 解决方案 -> 最小实现 -> 运行验证 -> 下一步缺口`.
- Add the smallest mechanism that solves the current chapter pain point.
